# 💰 Python Personal Expense Tracker

A Python-based interactive web application developed using **Streamlit** to record and manage daily expenses. The application allows users to add, search, delete, and categorize expenses while providing a summary of total spending through a simple web interface.

## 🚀 Live Demo

Try the application here:  
https://personal-expense-tracker-9ajsapkcttegtxvsniyzug.streamlit.app/

---

## 📌 Project Overview

The Python Personal Expense Tracker simplifies personal expense management. Users can record daily expenses, organize them into different categories, search expense records, delete expenses, and monitor their overall spending in one place.

This project was developed to strengthen my Python concepts by applying them in a practical portfolio project.

---

## ✨ Features

- Add new expenses
- Display all expenses
- Search expenses by Expense ID
- Delete expense records
- Calculate total expenses
- View category-wise expense summary
- Interactive web interface using Streamlit

## ▶️ How to Run the Project

```bash
git clone https://github.com/alihassan-ashraf/python-personal-expense-tracker.git
cd python-personal-expense-tracker
pip install -r requirements.txt
streamlit run app.py
```
